Автор: Евгений Екатериненко

Программа написана на языке программирования: С

Изначально игра ищет файл сохранения: SaveGame.sav в своем каталоге,
Если такой файл присутствует - то игра спросит желает ли игрок продолжить
и надо дать 2 варианта ответа (y/n)
Если игрок желает продолжить то игра загрузится с последнего сохранения
Сохранение происходит после каждого хода.

Если игрок не желает продолжить то будет создана новая игра. При создании
игры надо будет выбрать несколько параметров:
1. Выбрать тип игры (с компьютером или на 2 игрока)
2. Если игра с компьютером, то выбрать один из уровней сложности
(легкий, средний и сложный)
Далее начинается игра.
Для того что бы установить крестик или нолик на поле, надо указать координаты
Координата Х(столбец) - буква англ языка (A, B или C)
Координата Y(ряд) - число от 1 до 3.

Игра будет продолжена после зактытия этого файла.
